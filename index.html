测试mcp项目

这是一个用于测试GitHub MCP功能的Git仓库。

Git基本操作

  • 初始化仓库: git init
  • 添加文件: git add 文件名
  • 提交更改: git commit -m "提交信息"
  • 创建分支: git branch 分支名
  • 切换分支: git checkout 分支名
  • 合并分支: git merge 分支名

项目结构

  • README.md: 项目说明文件
  • .gitignore: Git忽略文件配置
  • example.txt: 示例文本文件
  • feature.txt: 特性分支演示文件
  • index.html: 项目网页文件（当前文件）

分支说明

  • main: 主分支，包含稳定代码
  • feature-branch: 特性分支，用于开发新功能
  • develop: 开发分支，用于日常开发工作
